# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-30

### Initial Release
- **Core AI**: Smart Model Selection (Auto-detects Flash/Pro/Preview models to avoid quotas).
- **Voice**: Integrated ElevenLabs TTS for realistic voice synthesis.
- **Vision**: Added screen capture and analysis capabilities.
- **Automation**: Implemented hybrid desktop automation (mouse, keyboard, shell, browser).
- **UI/UX**: Interactive "floating" desktop interface with PixiJS character.
- **Settings**: Secure API key management with "Show/Hide" toggle.
- **Security**: encrypted local credential storage.
- **Build**: Automated Windows installer generation via GitHub Actions.
